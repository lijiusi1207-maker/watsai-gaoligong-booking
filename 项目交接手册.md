## 3. 系统架构
单进程 Node 后端，同时托管**静态页面**与 **API**：
- **双模运行**：前端启动探测 `GET /api/ping`，通 → API 模式（真后端）；不通 → `localStorage` 回退（纯演示，后台/实时失效）
- **事件委托**：所有按钮用 `data-act="..."` + 单个 `document.addEventListener('click', …)`，**0 个 inline onclick**（微信安全）
- **持久化**：`data.json`（运行时生成；文件不存在时由 `defaultDB()` 自动重生，重生内容已含完整研学计划）
- **实时**：SSE 端点 `GET /api/stream?token=admin888` 向管理后台推送刷新
- **认证**：管理类接口需 `ADMIN_TOKEN`（默认仅用于本地回归，定义在 `服务器.js`）
- **身份证校验**：GB11643 18 位校验和，自动算年龄 / 性别；未成年 <18 需监护人；老年 ≥70 特殊标记
- **监听**：`server.listen(PORT, '0.0.0.0')`，`PORT = process.env.PORT || 8090`
